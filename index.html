核心能力

一台服务器，一个窗口，全部搞定

基于 Electron + ssh2 + xterm.js + Monaco Editor，为日常 Linux 运维打造的轻量工具。

稳定 SSH 终端

基于 ssh2 构建，支持 vim、top、tmux 等全屏程序，会话状态实时反馈，断连有提示。

SFTP 文件树

左侧可视化浏览远程目录，支持新建、重命名、删除、下载，目录操作一目了然。

在线文本编辑

内置 Monaco 编辑器，支持 30+ 语言高亮、多标签页、Ctrl+S 直接保存回服务器。

图片即时预览

远程图片直接打开查看，文本与图片预览均带本地缓存，重复打开更快。

拖拽上传

从 Windows 资源管理器直接把文件拖入左侧文件树，上传完成即可在远程使用。

资源实时监控

底部状态栏实时展示 CPU、内存、磁盘、负载，服务器状态随时掌握。

  • 左侧文件树，右侧终端 + 编辑器
  • 统一主题切换，界面、终端、编辑器配色一致
